### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

### 三、功能描述
系统分为（身份）：员工用户、管理员。

**员工用户功能：**
1. 登录
2. 首页
3. 个人中心（修改密码、个人信息）
4. 供应商信息管理
5. 药品管理
6. 进货管理
7. 销售管理



**管理员功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 基础数据管理（供应商类型、供应商信用等级类型、药品类型管理等）
4. 供应商信息管理
5. 药品管理
6. 进货管理
7. 销售管理
8. 员工管理
